H²GF-Net

Hybrid-scale Heterogeneous Graph Fusion for Multimodal Intent Recognition

Abstract

Multimodal Intent Recognition (MIR) aims to infer a speaker’s communicative intent by integrating heterogeneous textual, acoustic, and visual signals. However, intent-related cues are often distributed across fine-grained local variations and global semantic representations, making it challenging for existing approaches to explicitly model cross-modal and local–global dependencies within a unified framework. To address this challenge, we propose the Hybrid-scale Heterogeneous Graph Fusion Network (H²GF-Net), which integrates affective semantic prior-guided local key information selection with progressive hybrid-scale heterogeneous graph modeling for multimodal intent understanding.
